# pusher

此仓库用于转运docker镜像至阿里云仓库。

该仓库fork于[docker_image_pusher](https://github.com/tech-shrimp/docker_image_pusher)。

## 简介

使用Github Action将国外的Docker镜像转存到阿里云私有仓库，供国内服务器使用，免费易用

支持DockerHub, gcr.io, k8s.io, ghcr.io等任意仓库
支持最大40GB的大型镜像
使用阿里云的官方线路，速度快
